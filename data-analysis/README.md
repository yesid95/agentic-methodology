# Data Analysis Workflows

The data analysis domain will define structured workflows for human-led analytical work with specialized AI assistance.

## Purpose

This domain focuses on analysis planning, data interpretation, validation, and reporting while preserving human judgment over assumptions, conclusions, and publication.

## Expected Future Roles

- Analysis planning support
- Data validation support
- Interpretation support
- Report drafting support

## Intended Workflows

- Clearly scoped analytical questions
- Separation between exploration, validation, and interpretation
- Human review of assumptions and conclusions
- Traceable analytical outputs

## Current Status

Foundational structure only. Roles, protocols, prompts, and templates have not yet been defined.
